## Overview

A fully interactive terminal-style portfolio that runs entirely in the browser вҖ” no frameworks, no build tools, no dependencies. Visitors navigate the portfolio by typing commands (or clicking quick-access chips), exploring projects, experience, skills, and contact info just like they would in a real shell.

---

## Features

- **Interactive Terminal UI** вҖ” Type commands to explore content; supports Tab autocomplete, вҶ‘вҶ“ command history, and instant output rendering
- **Animated ASCII Banner** вҖ” Glowing header rendered in pure CSS with a pulse keyframe animation
- **Particle Network Background** вҖ” Canvas-based floating particle mesh with dynamic line connections
- **Custom Cursor** вҖ” Smooth lag cursor + dot combo with hover expansion states
- **CRT Scanline Overlay** вҖ” Subtle CSS repeating-gradient scanline effect for that authentic terminal feel
- **Live Clock** вҖ” Displays real-time `HH:MM:SS` in the title bar
- **Status Bar** вҖ” Sticky bottom bar showing command count, last command, and online indicator
- **Autocomplete Dropdown** вҖ” Fuzzy prefix-matched suggestions with keyboard navigation
- **Theme Toggle** вҖ” Switch between dark (default) and light mode via `theme -light` / `theme -dark`
- **Responsive** вҖ” Mobile-friendly layout with cursor and scanline effects gracefully disabled on touch devices

## Commands

| Command | Description |
|---|---|
| `about` | Short bio вҖ” who I am and what I do |
| `projects` | All projects with tech stack, dates, and GitHub links |
| `skills` | Full tech stack grouped by category |
| `experience` | Internship and work history |
| `education` | PDEU B.Tech details |
| `contact` | Email, phone, GitHub, location |
| `documentation` | Notion docs and notes |
| `help` | Lists all available commands |
| `clear` | Clears the terminal output |
| `theme -light` | Switch to light theme |
| `theme -dark` | Switch back to dark theme |

**Keyboard shortcuts:**
- `Tab` вҖ” autocomplete current input
- `вҶ‘` / `вҶ“` вҖ” cycle through command history
- `Escape` вҖ” dismiss autocomplete dropdown
- `Enter` вҖ” run command

## Tech Stack

This is a **zero-dependency, zero-build** project.

- **HTML5** вҖ” semantic structure and canvas element
- **CSS3** вҖ” custom properties, keyframe animations, CRT scanline via `repeating-linear-gradient`, sticky bars, responsive layout
- **Vanilla JavaScript** вҖ” terminal engine, command dispatcher, autocomplete, Canvas 2D particle system, requestAnimationFrame cursor loop

No React. No Webpack. No npm. Just one `.html` file.

## Customization

All content lives inside the `cmds` object in the `<script>` block. To update your own details:

```js
const cmds = {
  about: `...your HTML string...`,
  projects: `...your HTML string...`,
  // etc.
};
```

Colors and theming are controlled via CSS custom properties at the top of the `<style>` block:

```css
:root {
  --bg: #060a06;
  --g1: #00ff88;   /* primary accent */
  --y:  #f5e642;   /* command yellow */
  --b:  #4db8ff;   /* link blue */
  /* ... */
}
```
